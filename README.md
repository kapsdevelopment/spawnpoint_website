# spawnpoint_website

Static first version of the Spawnpoint marketing page. English is the default
language, with Norwegian preserved behind the language toggle.

## Local preview

Open `index.html` directly in a browser, or run:

```sh
python3 -m http.server 8123 --bind 127.0.0.1
```

Then visit `http://127.0.0.1:8123/`.
